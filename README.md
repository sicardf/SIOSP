## Description

A simple application in Swift 4.2 using the PJSIP library.

## Run

**SIOSP** use cocoapods for dependency management.
Run the following command :

```ruby
$ pod install
```

## Project implementation

#### Requirements
- iOS 12.0+
- Xcode 10+
- Swift 4.2+
- Cocoapoads 1.2+

#### Library
- [pjsip](https://github.com/chebur/pjsip)
